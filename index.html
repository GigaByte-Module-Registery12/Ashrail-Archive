The Game: Ashrail Archive

Rebuild memory. Restore the code.

Game Short description: Ashrail Archive is a console-text-based, text-adventure set in a post‑apocalyptic world where salvaged knowledge is survival. Play as a lone courier who follows clues, converses with eccentric survivors, and cracks a three‑word key to gain access to The Archive — a repository of records that can restart communities.

Built in C++ for a simple single-file compilation model (main.cpp) with helper header files, the game focuses on narrative choices, light resource management, and readable, modular code suitable

    Key Features:
  • Branching scenes with two choices per scene, single ending that reflects your path.
  • NPC side interactions with narrative effects and small gameplay.
  • Beginner-friendly C++ structure: one main.cpp plus .h headers — no extra .cpp files.
  • Simple player state: health, morale, supplies, and NPC encounter flags.
